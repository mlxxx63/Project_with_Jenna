General Usage

The aim of this program is to allow you, your friends & family to easily share your favourite songs and artists with each other. Our program allows you to share these playlists (no matter what music programs everyone uses) so your friends can easily give your music tastes a try. You will be able to add Youtube links to each song so you can easily find them, allowing you to listen to them as well.

Functions:

There is 8 options in total to choose from.

  • Option 1: Create playlists
  • Option 2: Show playlists
  • Option 3: Add new songs to a playlist
  • Option 4: Remove songs from a playlist
  • Option 5: Edit playlist details
  • Option 6: Save playlists to a file
  • Option 7: clear all songs from a playlist
  • Option 8: Exit

Using Functions:

How to use the functions and how they work.

Option 1: Create playlists

When the Main Menu asks "Enter what you want to do" you will type 1 and then hit the enter key on your keyboard. The program will then ask you how many playlists you would like to create. Type in the number you would like and then hit the enter key. It will then ask you for the name of your playlist. This is where you type in the name and again hit the enter key. Afterwards it will ask you for the genre of the playlist. You will again type out the genre and hit the enter key. The program will get you to repeat these steps until it has finished adding all the playlists you wanted to add. After adding all the playlists, it will print out the playlists you created including the name and number of the playlist (each playlist is given a number so it can be found easily). Function 1 ends and you will be prompted with the Main Menu again.

Option 2: Show playlists

When the Main Menu asks "Enter what you want to do" you will type 2 and then hit the enter key on your keyboard. The program will then list out the playlist(s) you have created including the name and genre of each playlist. If you have added any songs to the playlist, it will also list out the songs, showing you the song name, artist name and YouTube link. If you haven't added any songs yet it will say: "No songs added to this playlist yet!" You will then be returned to the main menu.

Option 3: Add new songs to a playlist

When the Main Menu asks "Enter what you want to do" you will type 3 and then hit the enter key on your keyboard. The program will ask you which playlist you want to add to and how many songs. First type in the playlist number and hit the enter key. Then type the number of songs you wish to add and click the enter key again. Answer all of the questions prompted to you and continue to the next one by hitting the enter key. The questions will ask you for the artist name, song name and then the song link. The song cannont be added if any of these questions are left blank. You will be returned to the main menu.

In order to share music with your friends, you will have to share a link to the song you wish to share. The link must be a youtube link. Without a Youtube link, the song won't be added as this ensures the artists copyrights are being protected.

Option 4: Remove songs from a playlist

When the Main Menu asks "Enter what you want to do" you will type 4 and then hit the enter key on your keyboard. The program will ask you which playlist you want to remove songs from. You will type in the number of the playlist and hit enter. It will then print out the list of songs inside of the playlist in a numbered list. You will type the number from the list of the song you want to remove. Hit enter. The program will then remove the song. Afterwards you will be returned to the main menu.

You will then answer the questions that allows you to rename your playlist and change the genre. Press enter after entering in new information. Playlist will update and you will retutn to main menu.

Option 8: Exit

When the Main Menu asks "Enter what you want to do" you will type 8 and then hit the enter key on your keyboard. Afterwards the program will stop running (exiting the program).

Getting Started:

The program will prompt you with all of the Functions available in a Main Menu.

It will look like this:


It will also ask you what you want to do next.

This is where you type in your command of what you want the program to do. You will type in any of the option numbers from 1-8.

The program will then ask you questions based on the function you are using.

    Ex. If you chose Option 1 you will then be asked how many playlists you would like to make. It will then ask you for the name and genre of each playlist.

After using a function, the program will again prompt you with the Main Menu. It will continue doing this and you can continue using the program until you choose Option 8 which will allow you to exit the program.
